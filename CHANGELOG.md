## [0.1.0] - 2026-07-19

### Added

- Typer and Rich CLI foundation with clean error handling and debug mode.
- `version`, `about`, and `doctor` core commands.
- Cross-platform CPU, memory, disk, and system summaries.
- Read-only file metadata, streaming hashes, trees, and largest-file discovery.
- JSON, Base64, UUID4, and cryptographically secure password utilities.
- Validated persistent config, standard/Hacker themes, and accessible effects.
- robots.txt-aware public web fetch, title, and link inspection.
- URL, redirect, peer-IP, timeout, content-type, and response-size safeguards.
- Automated unit and CLI tests across supported platforms and Python versions.
